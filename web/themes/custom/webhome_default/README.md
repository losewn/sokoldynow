webhouse SASS compilator

**Steps to prepare scss compilation**:

$ nvm install 14

$ nvm use 14

$ npm i

$ npm i -g gulp

$ npm i node-sass


or (one command on bash or shell)

$ nvm install 14 && nvm use 14 && npm i && npm i -g gulp && npm i node-sass

**task**

$ gulp watch - (watching edit scss files)
